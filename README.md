# Eggplosion
A 2D Pygame for the AI (1) Course Project.

This is a two-player (Human vs. Agent), turn-based strategy game on a grid board.
Each player controls one chicken that moves around the board and lay eggs on tiles to score points and can place turds (obstacles) a limited number of turds can be placed by each chicken to obstruct or block the opponent.

However, the board contains hidden trapdoors. If a chicken steps on a trapdoor, it becomes revealed, and the player receives a penalty.

The objective is to win by laying more eggs than the opponent before the move limit is reached.
